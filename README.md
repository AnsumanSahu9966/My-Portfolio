# Personal Portfolio Website

A modern, fully responsive portfolio website designed to present my academic background, technical expertise, projects, certifications, and professional achievements in a clean and engaging manner. The application combines contemporary web design principles with interactive user experience features to create a polished digital portfolio suitable for recruiters, collaborators, and fellow developers.

The website has been developed with an emphasis on accessibility, responsiveness, performance, and maintainable code, ensuring a consistent experience across desktop, tablet, and mobile devices.

## Live Demo

**Website:** https://ansuman-sahu-portfolio-263686015344.asia-southeast1.run.app/

## Features

* Modern, responsive user interface
* Smooth animations and transitions
* Dark/Light theme toggle
* Interactive project showcase
* Dedicated sections for education, skills, certifications, and achievements
* Downloadable résumé
* Functional contact form
* Mobile-first design with cross-browser compatibility
* Clean, structured, and maintainable codebase

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Web Design
* CSS Animations
* DOM Manipulation

## Purpose

This project serves as a comprehensive digital portfolio, consolidating my technical work, academic accomplishments, certifications, and professional development into a single platform. It reflects both my front-end development capabilities and my commitment to producing software that is intuitive, accessible, and visually refined.
